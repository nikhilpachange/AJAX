# Weather Dashboard

A simple weather application that fetches weather data for a specific city using the OpenWeather API.

## Features

- Get current weather information for any city.

## How to Run

1. Clone this repository or download the ZIP file.
2. Open `index.html` in your browser.
3. Enter a city name and click the "Get Weather" button.

### API Key

- Replace `YOUR_API_KEY` in `script.js` with your [OpenWeather API key](https://openweathermap.org/api).

## Technologies Used

- HTML
- CSS
- JavaScript
- AJAX
